# How To Install Visual Studio For UE 5.3
## 2025-12-15
### ![](https://raw.githubusercontent.com/DavidCai1874/my-tech-art-station-assets-storage-01/main/20251215174727.png)


The Unreal Engine version I use is **5.3.2**.  

***

1\. Before installing, **make sure there's no existing Visual Studio 2022** on your PC. 

2\. Open Unreal Editor, create a Blank project, and instead of a Blueprint project, **create a C++ project**. Unreal Editor should ask you to install Visual Studio 2022.

![](https://raw.githubusercontent.com/DavidCai1874/my-tech-art-station-assets-storage-01/main/20251215174727.png)

3\. In the Visual Studio Installer, go to **Modify**.

![](https://raw.githubusercontent.com/DavidCai1874/my-tech-art-station-assets-storage-01/main/20251215174956.png)

4\. Search for **"msvc"** and **uncheck the "(latest)" option**, pick an older version. I **choose 14.37** here.

![](https://raw.githubusercontent.com/DavidCai1874/my-tech-art-station-assets-storage-01/main/20251215180243.png)

5\. Search for **".net"** and check **both options for version 4.8**.

![](https://raw.githubusercontent.com/DavidCai1874/my-tech-art-station-assets-storage-01/main/20251215180439.png)

6\. Install Visual Studio 2022.

7\. Although we deselected the latest version and selected an older version for MSVC's installation, VS still installs two versions. So go to (assuming you didn't change the default installation path):

C:\Program Files\Microsoft Visual Studio\2022\Community\VC\Tools\MSVC. 

**Delete the other version and leave only the folder of 14.37**.

![](https://raw.githubusercontent.com/DavidCai1874/my-tech-art-station-assets-storage-01/main/20251215180920.png)

8\. Now restart Unreal Editor, you should be able to start a C++ project successfully.

![](https://raw.githubusercontent.com/DavidCai1874/my-tech-art-station-assets-storage-01/main/20251215181531.png)
